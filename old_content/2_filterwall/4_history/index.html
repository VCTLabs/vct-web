FilterWall Version History Highlights

Version 3.4

  • Added International Support
Version 3.3

  • More FilterWall hardware supported
Version 3.2

  • Added Quality of Services(QOS) Support
Version 3.1

  • DNS proxy
  • URL analysis utility
  • Auto list updates
  • Multi-user admin interface
  • Global Enable/Disable switch added to web cache interface
  • Improved Algorithm for Handling "ADVISE" Policy
Version 3.0

  • Improved Network Performance
  • Multiple Categories
  • Multiple Policies (Block, Advise, Safe, Log)
  • Improved Search Filter Algorithm
  • Block Review
  • DMOZ Content Database
  • Static NAT
  • Port Forwarding
  • Network Aliases
  • Improved Access Control
Version 2.1
  • Config Backup - Under the console you can backup or restore the preferences of the FilterWall to a floppy.
  • Web Caching Policies - Allows you to turn caching off or turn caching off to certain domains.
  • Enable search filter - Allows you to turn on and off the search filter.
  • Safe Domains - Allows you to add domains that are safe and should never be filtered.
  • Keywords - An update that allows you to remove our default keywords from the list.
  • Search Terms - An update that allows you to remove our default search terms from the list.
Version 2.0

  • Improved network performance
  • Stateful firewall
  • Windows user name logging
  • Fast recovery from power loss
  • Filter outgoing traffic by port, IP, or both
  • Display log entries for specific IP or user
  • New look for web administration
